# Women's Free Flight Film Festival 2027

Public marketing & information site for the Women's Free Flight Film Festival — a film festival celebrating women in free flight (paragliding, hang gliding, speedflying, hike-and-fly, and more).

## About
- **Submissions:** short films 2–20 minutes featuring a woman in free flight
- **Prizes:** up to $1,000
- **Deadline:** May 2027
- **Eligibility:** U.S.-based pilots

## Site
Static single-page site (`index.html`) hosted on GitHub Pages. No build step.

## To update
Edit `index.html` and commit. Placeholders to fill in:
- `FILMFREEWAY_URL` — your FilmFreeway submission page
- `CONTACT_EMAIL` — festival contact email
- `INSTAGRAM_URL` / `@yourhandle` — social handle

## Local preview
Open `index.html` in any browser.
